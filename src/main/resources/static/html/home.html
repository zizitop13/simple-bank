О демонстрации

О программе

Небольшая демонстрация возможностей Angular Js версии 1.7.2 и Spring Boot версии 2.0.4. на примере простого банковского приложения

Клиенты банка

На странице "Клиенты банка" отображаются в списке все клиенты банка, а также есть возможность для добавления нового клиента и редактирования счетов клиента.

Редактирование счетов клиента

На данной странице отображаются все счета выбранного клиента, есть возможность добавить счет и перейти на страницу переводов средств с выбранного счета на любой другой счет.

Перевод между счетами

На данной странице отображаются все счета клиента (кроме выбранного), есть возможность перевести сумму на другой счет клиента, а также найти и выбрать счет другого клиента банка.

Транзакции

На данной странице отображаются все транзакции, совершенные пользователями. Есть возможно применить фильтры.
